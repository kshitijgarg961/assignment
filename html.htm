HTML


HTML stands for hyper text markup language.With HTML you can create amazing web pages.
HTML describes the structure of a Web page.
HTML consists of a series of elements.
HTML elements tell the browser how to display the content.
HTML elements are represented by tags
You will enjoy them!

Following are some of the html elements-

HTML tag: It is the root of the html document which is used to specify that the document is html.


Head tag: Head tag is used to contain all the head element in the html file. It contains the title, style, meta, … etc tag.


Body tag: It is used to define the body of html document. It contains image, tables, lists, … etc.


Heading tag: It is used to define the heading of html document.


Bold tag: It is used to specify bold content in html document.


Italic tag: It is used to write the content in italic format.


Small (text) tag: It is used to set the small font size of the content.


Underline tag: It is used to set the content underline.


Deleted text tag: It is used to represent as deleted text. It cross the text content.


Anchor tag: It is used to link one page to another page.


Comment tag: It is used to set the comment in html document. It is not visible on the browser.


Font tag: It is used to specify the font size, font color and font-family in html document.


Empty (Non-Container) Tags:


Line break tag: It is used to break the line.

Image tag: It is used to add image element in html document.


Link tag: It is used to link the content from external source.


Meta tag: It is used to specify the page description. For example: last modifier, authors, … etc.


progress: Display the progress of an action, such as with a script.


q: Quote from some source.


rb: For Ruby annotations.


rp: Specify text to be displayed when Ruby annotations are not supported by the browser.


rt: Text component of a Ruby annotation.


iframe: Makes it possible to embed another page within a page. In HTML5 this is known as nested browsing.


img: For displaying images on a webpage. 


input: Input is used with various attributes to create form input elements.


ins: Useful for denoting text that has been added to an HTML file since the original version. Updates to a blog post for example. This is typically rendered 
as underlined text, although this can really be rendered just about any way you want with a stylesheet.


data: Used for script friendly data.


datalist: New in HTML5. Specify predefined data for an input.


dd: For description lists. Use with dt and dl tags.


del: Mark text as deleted without actually deleting it. This will typically be rendered as a strikethrough.


details: New in HTML5. Can be used for additional details which can be hidden or revealed.


dfn: Use when first defining a term.


dialog: Dialog box.


div: A divider or a kind of generic container.


s: For marking up text that is not longer relevant, applicable, or accurate.


samp: Sample output one might get from a particular computer program.


script: Scripts inside of an HTML document are enclosed in the script tags.


section: A nonspecific means of breaking up content within a webpage into sections.


select: Used with option tag for selecting a particular option.


small: Typically used for small print. Useful for comments within the main content.


source: Used with audio and video tags for specifying media source.


span: Useful for applying styles to or around text, especially inline.


strong: The enclosed text is of strong importance.


style: For declaring style sheets within a document.


sub: For displaying subscript. Useful for math, especially in conjunction with sup tag (superscript).


summary: Used in conjunction with the details tag for specifying a summary.


sup: For displaying superscript. Useful for math, especially in conjunction with sub tag(subscript).
